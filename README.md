## ⚙️ How to Run C++ Code (VS Code)

### 1. Install Requirements

* Install VS Code
* Install C++ Compiler (g++ via MinGW)

---

### 2. Compile the Code

Open terminal in VS Code and run:

```bash
g++ filename.cpp -o output
```

---

### 3. Run the Executable

```bash
./output
```

(On Windows)

```bash
output.exe
```

---

### 📌 Example

```bash
g++ array_declaration.cpp -o array
array.exe
```

---

### ⚠️ Notes

* Make sure `g++` is installed and added to PATH
* File name should match exactly
* Use `.exe` on Windows
